# kubernetes-sql
SQL using Kubernetes style

## Maven users

Add this dependency to your project's POM:

```xml
<dependency>
  <groupId>com.github.kubesys</groupId>
  <artifactId>kubernetes-sql</artifactId>
  <version>5.0.0</version> 
</dependency>
